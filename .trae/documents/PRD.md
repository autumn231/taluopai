# 塔罗秘境 · Mystic Tarot PRD

## 1. 产品概述

「塔罗秘境」是一款融合神秘主义美学与现代交互体验的在线塔罗占卜单页应用，以深邃的宇宙星辰为视觉主题，提供三种经典的塔罗牌阵（单张牌阵 / 三张牌阵 / 凯尔特十字）与完整的 78 张塔罗牌图鉴，让用户通过沉浸式的仪式感体验获得心灵指引。

- **核心问题**：传统塔罗占卜 App 视觉粗糙、缺乏沉浸感，无法传递塔罗的"神圣仪式感"
- **目标用户**：对神秘学 / 自我探索感兴趣的大众人群（18-35 岁），寻求心灵慰藉、决策参考或娱乐体验
- **市场价值**：以"内容 + 视觉 + 仪式感"差异化为核心，建立情感连接，提升用户停留时长

## 2. 核心功能

### 2.1 功能模块

1. **首页 / 仪式入口**：星河流转的 Hero 区域、三种牌阵选择、占卜须知
2. **占卜流程页**：冥想环节 → 洗牌动画 → 选牌交互 → 翻牌揭晓
3. **解读结果页**：所选牌位的正逆位解读、整体牌阵综合解读、关键词、建议
4. **塔罗图鉴页**：78 张牌（大阿尔卡那 22 + 小阿尔卡那 56）的图鉴浏览与详情查看
5. **历史记录页**：本地存储最近 10 次占卜记录，可重新查看

## 3. 核心流程

```
用户进入首页
   ↓
选择牌阵（单张 / 三张 / 凯尔特十字）
   ↓
进入冥想引导（5 秒倒计时）
   ↓
进入洗牌台：用户拖动鼠标或点击"开始洗牌"触发洗牌动画
   ↓
进入选牌台：按牌阵数量从牌堆中抽牌
   ↓
进入翻牌台：依次翻开每张牌
   ↓
进入结果页：展示每张牌解读 + 综合解读
   ↓
可选：保存到历史 / 重新占卜 / 分享
```

## 4. 用户界面设计

### 4.1 设计风格

- **整体调性**：维多利亚神秘学 × 宇宙星辰，融合古典仪式感与未来感
- **主色调**：
  - 深夜蓝紫 `#0a0824`（主背景）
  - 紫黑 `#1a0f3d`（次背景）
  - 神秘金 `#d4af37`（主点缀）
  - 银白月色 `#e8e4f3`（正文）
  - 玫瑰金 `#e91e63`（强调）
- **按钮样式**：双线描金边框、悬浮时填充金色光晕、点击时涟漪扩散
- **字体方案**：
  - 标题：`Cinzel Decorative`（古罗马碑文感）
  - 副标题：`Cormorant Garamond`（优雅衬线）
  - 正文：`Noto Serif SC`（中文衬线）
  - UI 文字：`Inter`（仅用于按钮、标签等小字）
- **布局风格**：以居中对称的"仪式台"为意象，关键内容卡片化悬浮
- **图标风格**：线性细描金，统一 1.5px 描边，使用 `lucide-react`

### 4.2 页面设计概览

| 页面 | 模块 | UI 元素 |

| 占卜页 | 冥想引导 | 圆形符文旋转 + 中央倒计时数字 + 底部引导语 |
| 占卜页 | 洗牌台 | 78 张扇形展开，鼠标视差跟随 |

### 4.3 响应式

- **桌面优先**（≥1024px）：完整布局，多列网格
- **平板**（768-1023px）：网格降为 2 列，牌阵选择上下排布
- **手机**（<768px）：单列布局，牌阵选择垂直堆叠，导航折叠为汉堡菜单，触摸优化（点击区 ≥ 44px）
- **横屏适配**：在 mobile 横屏下占卜流程关键步骤居中显示

### 4.4 动态场景指引

- **环境氛围**：深邃宇宙背景，缓慢漂移的星云，闪烁的星点
- **粒子系统**：150+ 颗星点随机分布，鼠标移动时附近粒子受引力牵引产生光弧
- **光效**：金色光晕（`box-shadow` + 模糊滤镜）、旋转的符文圆环
- **关键动效**：
  - 标题渐入 + 字符逐个出现
  - 卡片悬停 3D 倾斜（perspective + rotateX/Y）
  - 翻牌 3D 旋转（rotateY 0→180deg + 背面 visibility 切换）
  - 抽牌：选中牌从牌堆弹出，划过弧线落入牌位
  - 揭晓：牌位逐个发金光后翻转
  - 滚动：内容随滚动视差上移
- **缓动函数**：cubic-bezier(0.4, 0, 0.2, 1) 与 cubic-bezier(0.68, -0.55, 0.27, 1.55)（弹性）

## 5. 数据内容

- **78 张塔罗牌完整数据**：每张含中英文名、关键词、Upright / Reversed 含义、爱情/事业/财运三个维度的简要解读、象征图元素描述
- **3 种牌阵位置定义**：单张（核心指引）、三张（过去/现在/未来）、凯尔特十字（10 个位置的含义）
- **综合解读模板**：根据牌阵中牌的组合（正/逆位）生成衔接性解读文本

## 6. 部署要求

- 部署平台：**腾讯 EdgeOne Pages**
- 输出格式：纯静态资源（`vite build` 后 `dist/` 目录）
- 兼容性：现代浏览器（Chrome 90+ / Safari 14+ / Firefox 88+ / Edge 90+）
- 性能预算：首屏 LCP < 2.5s，移动端 60fps 流畅
